# Epilepsia por Colapso Neurotopológico: Código y Recursos

Este repositorio contiene el código y los recursos para el artículo: **"Colapso de la Complejidad en Crisis Epilépticas: Un Análisis Neurotopológico con Entropía, Homología y Curvatura de Ricci"** por Arturo Salazar Chon.

El objetivo de este trabajo es demostrar que la crisis epiléptica se caracteriza por un colapso de la complejidad dinámica y estructural del cerebro, un estado de orden patológico en lugar de desorden.

## Requisitos

Para ejecutar el código de este proyecto, es necesario tener las bibliotecas de Python especificadas en el archivo `requirements.txt`.

Para instalar todas las dependencias, clona este repositorio y ejecuta el siguiente comando en tu terminal:
```bash
pip install -r requirements.txt
